OPTION 1: DEPLOY WITH VERCEL (EASIEST - 2 MINUTES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to: https://vercel.com/new

2. Click "Import Project" → Connect GitHub → Select "mohaneni/samkass"

3. Configure:
   - Root Directory: kaasflow/backend
   - Click "Deploy"

4. After deploy: Settings → Environment Variables
   - Add all variables from kaasflow/backend/.env
   - (See DEPLOY_NOW.md for full list)

5. Deployments → Redeploy

6. Done! Your backend is live at: https://your-project.vercel.app

OPTION 2: DEPLOY WITH CLI (3 MINUTES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

# Install Vercel CLI
npm install -g vercel

# Login
vercel login

# Deploy
cd kaasflow/backend
vercel

# Add environment variables
vercel env add SUPABASE_URL
(add all variables from .env)

# Deploy to production
vercel --prod
