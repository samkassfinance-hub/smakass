✅Pre-Installation Checklist
    📖How to Test
    1. Check the System Status above
    2. Click "Test Install Prompt" button
    3. Check the Console Output for messages
    4. If no native prompt, an instruction modal will appear
    5. Follow the on-screen instructions to install
    ❓Troubleshooting
    • No status messages? → PWA module not loaded, refresh the page
    • Service Worker failed? → /sw.js not found or not accessible
    • Nothing happens on click? → Browser may not support PWA or app already installed
    • Modal appears instead of prompt? → Normal on iOS, follow the instructions
