📱 WhatsApp Automation Setup

Standalone version - No backend required!

Step 1: Save Your WhatsApp Settings

Format: +[country code][number] (e.g., +919876543210)

✅ Saved Settings

WhatsApp Number:

Business Name:

Saved in browser localStorage

Step 2: Install Python & PyWhatKit

📥 Install Python

  1. Download from: python.org/downloads
  2. During installation: ✅ Check "Add Python to PATH"
  3. Open terminal and verify:
    python --version

📦 Install PyWhatKit

Open terminal/command prompt and run:

pip install pywhatkit

This is the only package required for WhatsApp automation!

Step 3: Test WhatsApp Sending

🧪 Run Test Script

In terminal, navigate to backend folder and run:

cd kaasflow/backend
python test_whatsapp_direct.py +919876543210

What happens:

  • Script schedules a test message for 2 minutes from now
  • WhatsApp Web opens automatically in your browser
  • Message is sent to your number
  • You receive it on your phone

Step 4: Setup Daily Automation

📅 Windows Task Scheduler

  1. Open Task Scheduler (search in Start menu)
  2. Create Basic Task → Name: "WhatsApp Loan Reminders"
  3. Trigger: Daily at 9:00 AM
  4. Action: Start a program
    C:\path\to\kaasflow\backend\run_daily_reminders.bat
  5. Save and test

🐧 Linux/Mac (Cron)

Edit crontab:

crontab -e
# Add this line:
0 9 * * * /path/to/kaasflow/backend/run_daily_reminders.sh

⚠️ Important Requirements

  • WhatsApp Web: You must be logged into WhatsApp Web
  • Computer ON: Keep computer running at reminder time
  • Browser: Chrome or Firefox must be installed
  • Client Numbers: Must be valid WhatsApp numbers with country code
