  • name, phone
  • password_hash, is_active
  • created_at, updated_at, last_login

subscriptions
  • id (UUID, primary key)
  • user_id (foreign key)
  • plan_type, status
  • start_date, end_date, client_limit
  • created_at, updated_at

app_backups ← Main sync table
  • id (UUID, primary key)
  • user_email (unique)
  • clients_json, loans_json, payments_json, settings_json (JSONB)
  • created_at, updated_at

audit_logs
  • id (UUID, primary key)
  • user_id (foreign key)
  • action, table_name, record_id
  • changes (JSONB)
  • ip_address, user_agent
  • created_at
