```
                         │
                         ▼
              ┌────────────────────┐
              │  Loan Record       │
              │  • start_date      │
              │  • duration        │
              │  • type (monthly)  │
              └──────────┬─────────┘
                         │
                         ▼
              ┌────────────────────┐
              │  Payment Records   │
              │  (Count payments)  │
              └──────────┬─────────┘
                         │
                         ▼
              ┌────────────────────┐
              │  Calculate:        │
              │  next_due =        │
              │  start_date +      │
              │  (payments + 1) ×  │
              │  interval          │
              └──────────┬─────────┘
                         │
         ┌───────────────┼───────────────┐
         │               │               │
    [Monthly]       [Weekly]         [Daily]

         │               │               │
         └───────────────┼───────────────┘

                         ▼
              ┌────────────────────┐
              │  next_due_date     │

              └──────────┬─────────┘
                         │
                         ▼
              ┌────────────────────┐

              └──────────┬─────────┘
```

## ⚙️ System Architecture Layers

```
┌─────────────────────────────────────────────────────────┐
│                     LAYER 1: UI                          │
│  ┌────────────────────────────────────────────────┐     │
│  │  whatsapp-settings.html                        │     │
│  │  • Configure WhatsApp number                   │     │
│  │  • Set business name                           │     │
│  │  • Enable/disable reminders                    │     │
│  │  • Send test messages                          │     │
│  └────────────────────────────────────────────────┘     │
└──────────────────────────┬──────────────────────────────┘
                           │ HTTPS/REST
                           ▼
┌─────────────────────────────────────────────────────────┐
│                  LAYER 2: API                            │
│  ┌────────────────────────────────────────────────┐     │
│  │  routes/whatsapp_routes.py                     │     │
│  │  • GET  /api/whatsapp/settings                 │     │
│  │  • POST /api/whatsapp/settings                 │     │
│  │  • POST /api/whatsapp/test                     │     │
│  │  • GET  /api/whatsapp/logs                     │     │
│  └────────────────────────────────────────────────┘     │
└──────────────────────────┬──────────────────────────────┘
                           │ SQL Queries
                           ▼
┌─────────────────────────────────────────────────────────┐
│                LAYER 3: DATABASE                         │
│  ┌────────────────────────────────────────────────┐     │
│  │  Supabase (PostgreSQL)                         │     │
│  │  • kf_whatsapp_settings                        │     │
│  │  • kf_whatsapp_logs                            │     │
│  │  • kf_loans                                    │     │
│  │  • kf_clients                                  │     │
│  │  • kf_payments                                 │     │
│  └────────────────────────────────────────────────┘     │
└──────────────────────────┬──────────────────────────────┘
                           │ Daily Query
                           ▼
┌─────────────────────────────────────────────────────────┐
│              LAYER 4: AUTOMATION                         │
│  ┌────────────────────────────────────────────────┐     │
│  │  Scheduler (Task Scheduler / Cron)             │     │
│  │  Triggers: whatsapp_reminder.py                │     │
│  │  • Daily at 9:00 AM                            │     │
│  │  • Checks loans due today                      │     │
│  │  • Sends reminders via PyWhatKit               │     │
│  └────────────────────────────────────────────────┘     │
└──────────────────────────┬──────────────────────────────┘
                           │ API Calls
                           ▼
┌─────────────────────────────────────────────────────────┐
│              LAYER 5: MESSAGING                          │
│  ┌────────────────────────────────────────────────┐     │
│  │  PyWhatKit                                     │     │
│  │  • Opens WhatsApp Web                          │     │
│  │  • Sends messages                              │     │
│  │  • Handles browser automation                  │     │
│  └────────────────────────────────────────────────┘     │
└──────────────────────────┬──────────────────────────────┘
                           │
                           ▼
                    [Client WhatsApp] 📱
```
